# Timothy - Examen de grammaire française 🇫🇷

Un examen interactif de grammaire française pour Timothy (niveau CM2/6ème).

## 🚀 Live Demo

**[houenyang-momo.github.io/timothy-french-exam/](https://houenyang-momo.github.io/timothy-french-exam/)**

Ouvrir sur iPad → Ajouter à l'écran d'accueil pour une expérience application.

## 📚 Contenu — 5 sections, 30 questions

| Section | Sujet | Questions |
|---------|-------|-----------|
| **G1 — L'adjectif** | Définition, accord, placement, sens selon la place | 6 |
| **G4 — Compléments circonstanciels** | Lieu, temps, manière, cause — repérage et création | 6 |
| **G5 — Complément du nom** | Prépositions, articles contractés, composition | 6 |
| **G6 — COD / COI** | Complément d'objet direct et indirect, pronominalisation | 6 |
| **Futur Simple** | Terminaisons, auxiliaires, verbes réguliers et irréguliers | 6 |

## 🎯 Types d'exercices

- **Choix multiples** (définitions, règles)
- **Texte à trous** (conjugaison, accords)
- **Mots à sélectionner** (identifier dans une phrase)
- **Vrai/Faux** (compréhension des règles)

## ✨ Fonctionnalités

- ✅ Correction instantanée avec explications
- ✅ Barre de progression en temps réel
- ✅ Navigation libre entre les sections
- ✅ Sauvegarde automatique (localStorage)
- ✅ Résultats détaillés par section
- ✅ Rafraîchir / Recommencer
- ✅ Design responsive (iPad, tablette, mobile, desktop)

## 🛠 Développement

Fichier unique `index.html` — CSS et JS inline. Aucune dépendance externe. Fonctionne même hors ligne une fois chargé.

### Déploiement local

```bash
# Ouvrir directement dans le navigateur
open index.html
```

### Build / Déploiement

Le site est déployé via GitHub Pages. Pour mettre à jour :

```bash
git add index.html
git commit -m "mise à jour"
git push
```
